docker + laravel + swagger

2019-03-09
学电脑

文章目录

在docker中安装swagger，并在laravel中进行使用

  1. 本地安装docker
1

sudo apt install -y docker
  1. 拉取 swagger 镜像
1

docker pull swaggerapi/swagger-ui
  1. 启动 swagger 镜像
1

docker run -d -p 8081:8080 -e SWAGGER_JSON=/foo/api-docs.json -v /home/summer/workplace/union-account-system/storage/api-docs/:/foo swaggerapi/swagger-ui

说明 ：

-d 后台运行

-p 用主机的 8081端口接收 容器的 8080 端口【将容器中的 8080 端口映射到主机的 8081 端口

】

-e 设置环境变量 SWAGGER_JSON

-v 将 主机的 文档 目录映设到容器的中。环境变量设置的文件路径需要映射到文档在主机真实路径上

  1. 参考文档

  2. 在laravel中使用命令行生成 storage/api-docs.json
