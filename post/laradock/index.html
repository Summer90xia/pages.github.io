工作之余，对laradock的使用方法做了一个说明

仓库地址

https://github.com/laradock/laradock

操作流程

1、安装docker

1
2

sudo apt update
sudo apt install -y docker docker-compose

2、将docker作为非root用户运行，参考

3、进入到工作目录，从github上拉取已改进的laradock项目

1

git clone --branch member --depth 1 git@github.com:hk01-digital/dd01-laradock.git

4、进入到 dd01-laradock，执行 cp env-example .env

5、在 .env 文件中找到下列选项，并做修改

 1
 2

# 配置工作目录
APP_CODE_PATH_HOST = /home/xxx/projects

# 取消node，yarn的安装
WORKSPACE_INSTALL_NODE = false
WORKSPACE_INSTALL_YARN = false

# 安装 mongo
WORKSPACE_INSTALL_MONGO = true
PHP_FPM_INSTALL_MONGO = true

# mysql 使用 最新 版本
MYSQL_VERSION=latest

6、运行 docker

1

docker-compose up -d workspace nginx mysql redis mongo rabbitmq

若启动过程中出现问题，可以去掉 -d参数，查看具体错误信息，首次启动可能需要有点时间。如果修改了配置文件，记得要重新 docker-compose build

7、进入到docker工作区

1

docker exec -it --user=laradock laradock_workspace_1 bash

可能会遇到的问题

1、升级原有php-fpm容器中的php版本时肯能会遇到报错

1
2

E: Unable to fetch some archives, maybe run apt-get update or try with --fix-missing?
ERROR: Service 'php-fpm' failed to build: The command '/bin/sh -c if [ ${INSTALL_IMAGEMAGICK} = true ]; then     apt-get install -y libmagickwand-dev imagemagick &&     pecl install imagick &&     docker-php-ext-enable imagick ;fi' returned a non-zero code: 100

解决方案： docker-compose build --no-cache php-fpm

虚拟主机配置

1、修改nginx 配置文件

1

4

cd  dd01-laradock/nginx/sites
cp laravel.conf.example sass.dd01.com.conf
vim sass.dd01.com.conf

# 进入文件，修改
server_name sass.dd01.com;
root /var/www/member-union-account-system/public;

2、修改宿主机hosts文件

1

127.0.0.1 sass.dd01.com

3、重启 nginx 服务

1

docker-compose restart nginx

4、浏览器访问 http://sass.dd01.com 查看是否配置成功

使用技巧

1、停止容器

1

docker-compose down

2、查看容器错误日志

1

docker-compose logs nginx

#!/bin/bash

if [ -z $1 ] ; then 
	echo "参数缺失"
	exit 2
fi

case $1 in
"start")
	docker-compose up -d workspace php-fpm nginx mysql redis mongo rabbitmq;;
"workspace")
	docker exec -it --user=laradock laradock_workspace_1 bash;;
"mysql")
	docker exec -it laradock_mysql_1 bash;;
"redis")
	docker exec -it laradock_redis_1 bash;;
"*")
	echo "该值暂未匹配";;
esac
